BlueberryPeach for [ghostty](https://github.com/ghostty-org/ghostty).

## Installation

1. Copy the `blueberry_peach_*` files from this directory into your `~/.config/ghostty/themes` directory.
   - Alternatively, you can clone this repository and symlink the files.
2. Select the theme in your ghostty config file, usually at `~/.config/ghostty/config`.

   ```
   theme = dark:blueberry_peach_dark,light:blueberry_peach_light
   ```

3. Reload or restart Ghostty.

For more information, you can read the [Ghostty theme documentation](https://ghostty.org/docs/features/theme).
