# 🫐 Windows Terminal 🍑

BlueberryPeach for [Windows Terminal](https://learn.microsoft.com/en-us/windows/terminal/install).

## Installation

1. Open your Windows Terminal's Settings at `%LOCALAPPDATA%\Packages\Microsoft.WindowsTerminal_8wekyb3d8bbwe\LocalState\settings.json`
   - If it's no there, check [here](https://learn.microsoft.com/en-us/windows/terminal/install#settings-json-file) to find the location of your settings file
2. Add the schema from [`./blueberry_peach_dark.json`](./blueberry_peach_dark.json) and/or [`./blueberry_peach_light.json`](./blueberry_peach_light.json) to the `schemes` array
